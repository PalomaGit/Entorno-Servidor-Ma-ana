# Entorno Servidor 🖥️

Este proyecto es mi **entorno de servidor PHP dockerizado** para los ejercicios de clase.  
Me permite levantar un servidor local y trabajar con bases de datos sin complicaciones.

---

## 🛠️ Tecnologías

- PHP
- Docker
- Docker Compose
- MySQL / MariaDB
- Servidor web (Nginx / Apache)
